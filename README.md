## Description

This project detects human emotions using uploaded images and live webcam input. The system uses DeepFace and OpenCV to classify emotions such as Happy, Sad, Angry, Fear, Surprise, Disgust, and Neutral.

## Technologies Used

- Python
- DeepFace
- OpenCV
- TensorFlow
- Tkinter

